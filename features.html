Current Features

The game is currently in Pre-Alpha, which means there's still a lot of bugs and features that haven't yet made it into the game. If you want to report a bug or request a new feature, head on over to our GitHub page, and open a new issue.

v0.1

  • Temporary avatar with animations
  • Avatar walks, talks, and jumps
  • Support for multiplayer and in-game chat
  • Basic user interface with mobile capabilities
    • Date and time of game world
    • Metabolics: Mood, Health, Energy, iMG, Currants
    • Inventory (not currently usable)
    • Button for switching between desktop/mobile view
    • Chat panes: Global, Local, and Other
    • Settings buttons: Console, Bug Reports, Community, Volume, Fullscreen
    • Current location and Map (map not currently usable)
    • List and links to neighboring streets
  • Console commands for debugging and testing
  • Approximately 95% of original streets available for exploration
  • Integration of SoundCloud; temporary music plays in-game

Known Issues

Last updated 2014-02-25

  • Background images not displaying in some fullscreen browsers
  • Sound does not play on mobile browsers
  • Some graphical errors (mostly in the UI) when using older versions of Internet Explorer
  • Some streets (rare) do not have all art assets available and will not load
  • Minor animation errors with avatar
