What is Children of Ur?

Children of Ur is a community reboot of the now-defunct game Glitch™, a browser-based MMO created by Tiny Speck, Inc. The majority of the features found in the original game are still in development.

When can I play?

Right now, we're working on a lot of the behind-the-scenes stuff that makes the game run. We're adding new features pretty often, and you can check out the current state of the project on our demo page at any time. Since the game is still in development, you may find bugs or missing features in the demo (and if you do, we encourage you to report them by clicking on the little bug icon).

How can I follow along?

For more information about the project, updates, or contact information, follow us on Twitter or check out our Google+ community.
